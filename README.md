# Pranit Dhanade — Portfolio Website

A premium static portfolio built with only HTML5, CSS3, and Vanilla JavaScript. It is deployable directly to GitHub Pages with no build step.

## Features

- Dark glassmorphism interface with gradient lighting, grid/noise texture, and large editorial typography
- Canvas neural-network hero background and animated research graph
- Mouse parallax, magnetic buttons, custom cursor, spotlight hover, text reveal, scroll progress, and section reveals
- Responsive sections for About, Skills, Projects, Research, GitHub, Timeline, Contact, and Footer
- Live GitHub API integration for `pranitdhanade-sys`
- SEO, Open Graph, and Twitter card metadata

## Deploy to GitHub Pages

1. Push this repository to GitHub.
2. Open **Settings → Pages**.
3. Select the branch and root folder.
4. Save and wait for Pages to publish.

## Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
